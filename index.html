Integrasi Google Sheets (GAS Backend)

Langkah Integrasi:

  1. Klik Buat Google Sheets Baru.
  2. Pilih menu Ekstensi > Apps Script.
  3. Hapus seluruh kode default di editor Apps Script (file Code.gs).
  4. Klik Lihat Kode Apps Script di sini, salin kodenya, dan paste ke editor Apps Script.
  5. Klik Terapkan (Deploy) > Penerapan Baru > Aplikasi Web (Atur akses: Siapa Saja / Anyone).
  6. Salin URL Aplikasi Web yang diberikan ke kolom di bawah ini.
